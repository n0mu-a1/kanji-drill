# かん字ドリル 🖌️

小学1年〜6年の**教育漢字**を、**読み（ふりがな）**と**熟語・例文**の4択クイズで楽しく練習できる、スマホ対応のインタラクティブ Web アプリ。

## 特長

- **全6学年ミックス対応** … 学年（1〜6年）を選ぶか「ぜんぶミックス」で横断練習
- **3つの出題モード**
  - 🔤 **読み（ふりがな）** … 熟語を見て正しい読みを4択
  - 📖 **熟語・例文** … 例文の中の語の読みを文脈から4択（漢字を空欄ハイライト）
  - ✏️ **単語（訓読み）** … 漢字＋送り仮名の訓読み語を見て正しい読みを4択（例：学ぶ→まなぶ）
- **その場で解説** … 正解の読み・音訓・意味・熟語例を表示
- **演出** … 正誤サウンド、3連続正解で紙吹雪、💮はなまる、結果スタンプ＆ランク
- **復習** … 間違えた漢字を結果画面でまとめ表示
- **記録** … 学年×モード×問題数ごとのベストスコアを保存（localStorage）
- **オフライン動作 / PWA** … iPhone の「ホーム画面に追加」でアプリのように使える

## 収録データ

`kanji-data.js` … 学年別漢字配当表（2020年度）準拠の **計1013字**。
各字に「音読み・訓読み・意味・熟語2語（読み/意味付き）・例文（読み付き）」を収録。

| 学年 | 字数 |
|---|---|
| 1年 | 79 |
| 2年 | 160 |
| 3年 | 200 |
| 4年 | 199 |
| 5年 | 192 |
| 6年 | 183 |

> データは複数AIエージェントで学年別に生成し、別エージェントが**ふりがな・配当を敵対的に校正**したうえで、例文の漢字交じり表記を正規化しています。

## 使い方（ローカル）

```sh
cd kanji-drill
python3 -m http.server 8765
# ブラウザで http://localhost:8765/ を開く
```

`file://` で `index.html` を直接開いても動作します（フォントはオンライン時のみ Google Fonts を使用、オフラインはシステムフォントにフォールバック）。

## デプロイ

静的サイトなので Vercel 等にそのまま公開できます（`kanji-drill/` をルートに）。

## ファイル構成

```
kanji-drill/
├─ index.html              # アプリ本体（UI + クイズロジック、単一ファイル）
├─ kanji-data.js           # 漢字データ（window.KANJI_DATA）
├─ manifest.webmanifest    # PWA マニフェスト
├─ icon.svg                # アイコン
└─ README.md
```
